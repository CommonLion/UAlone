# 니혼자산다 (UAlone)

## 📌 주요 기능
- **펫닥터**
  - 반려동물의 피부 질환을 사진으로 간단하게 진단할 수 있는 기능.

- **트래킹**
  - 매일매일 섭취/배변/구토 등의 건강 상태를 기록할 수 있는 기능.

- **캘린더**
  - 반려동물 관련 다양한 일정을 편리하게 관리할 수 있는 기능.
 
- **커뮤니티**
  - 다양한 반려인들과 소통하며 다양한 건강 관리 조언을 주고받을 수 있는 기능.
